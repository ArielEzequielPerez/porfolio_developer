# 🌟 Portafolio de Ariel Perez

¡Bienvenido a mi portafolio! Este proyecto está diseñado para mostrar mis habilidades, proyectos y experiencia como desarrollador. 🚀

---

## 🛠️ Tecnologías Utilizadas
- **Framework**: Nuxt 3
- **Estilos**: Tailwind CSS
- **Formularios**: FormKit
- **Iconos**: Nuxt Icon
- **Linting**: ESLint

---

## 🚀 Despliegue Local

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

El proyecto estará disponible en `http://localhost:3000`.

---

## 🐳 Despliegue con Docker

También puedes ejecutar el proyecto utilizando Docker:

1. Construye la imagen:
   ```bash
   docker compose build
   ```

2. Levanta los contenedores:
   ```bash
   docker compose up -d
   ```

El proyecto estará disponible en `http://localhost:4000`.

## 📂 Estructura del Proyecto

```
├── assets/         # Archivos estáticos (CSS, imágenes, etc.)
├── components/     # Componentes reutilizables
├── layouts/        # Diseños base para las páginas
├── pages/          # Rutas y vistas del proyecto
├── plugins/        # Plugins de Nuxt
├── nuxt.config.ts  # Configuración principal de Nuxt
```
